Links and Images

The Web would not be the web without link and image elements.These elements, which allow you to reference resources outside of the document you are currently working on, represent the hyper of "hypertext".

Links

Link elements are defined by the <a> tag, which stands for anchor. Here's an example of a basic link tag. 

        
          <a href='https://developer.mozilla.org/en-US/docs/Web/HTML'>
            Mozilla Developer Network
          </a>

This is an absolute link; it always goes to the same location. HTML also allows you to define a relative link based on the current document. This example goes to the HTML Basics file that we saw earlier. 

        
          <a href='./html-basics.html'>
            HTML Basics
          </a>

In both cases, the href attribute controls the link's destination.

Images

Images work kind of like links, but instead of an href attribute, they have an src attribute for "source". It's also good to define an alt tag for accessibility purposes. 

        
          <img alt='what a cute doggo' src='http://i.imgur.com/mUIQRVy.jpg'> />

Animated GIFs work too! You treat them exactly the same.
