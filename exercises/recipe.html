Using the tags you learned about this week, construct a semantic recipe page from the text in the recipe-text.txt file in this folder.

You can place your recipe inside of this <body> tag so you don't have to re-create the entire page from scratch.

Consider using an <aside> element and an unordered list for the ingredients, an <article> tag for the main recipe with an ordered list of steps inside it, and don't forget to include an image!
