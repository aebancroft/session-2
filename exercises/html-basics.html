Body Text

This is the humble paragraph tag, possibly the element you will encounter the most.

Paragraph text, like headings, is a block-level element. Typically these elements will expand to fill all available space.

There is another category of elements, called inline elements. The bold text you see here is one example, it's generated by a <strong> tag.

How to you write the name of a tag instead of creating the tag in place? HTML allows you to use a special code starting with the & character to display these characters literally. Take a look at the source code of this page to see how it's done.

Common inline values include:

  • <strong>: for bold text
  • <em>: for emphasis
  • <span>: Span tags for arbitrary snippets
  • <sup> and <sub> for superscript and subscript, respectively.
  • There are Many more!

As you may have noticed by now, elements (both span and block) can appear nested inside one another. The paragraph above has a list element inside of it — this is totally fine!


List elements

Speaking of lists, here are a few examples:

Ordered Lists

  1. Use these when you need numbers, or when order is meaningful
  2. You can create this using the <ol> tag.
  3. Inside should go one or more list items (<li> tags).

Unordered lists

  • Use an unordered list when you don't want numbers, or when order is not as meaningful.
  • You use the <ul> tag to create these elements.
  • Unordered lists are often used in many other contexts as well, like in a site's navigational menus.
